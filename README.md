
# Russ AI 🥃
**The Intelligent Hospitality Engine**

Russ AI is a specialized AI intelligence platform designed for high-volume, high-end hospitality venues. It bridges the gap between massive inventory lists and real-time service expertise.

## 🚀 Core Features
- **The Sommelier Engine:** Vector-based alcohol recommendations using RAG (Retrieval-Augmented Generation).
- **The Slam-Meter:** Real-time kitchen and floor latency predictive analytics.
- **Russ-Script:** Automated, natural-language upselling scripts for front-of-house staff.

## 🛠 Tech Stack (2026 Enterprise Grade)
- **Backend:** Java 25 (LTS), Spring Boot 4.0.3
- **Database:** Amazon RDS (PostgreSQL 18) + pgvector 0.8.0
- **AI/ML:** Amazon Bedrock (Claude 4.6 Sonnet)
- **Infrastructure:** AWS Fargate (ECS), AWS Lambda
